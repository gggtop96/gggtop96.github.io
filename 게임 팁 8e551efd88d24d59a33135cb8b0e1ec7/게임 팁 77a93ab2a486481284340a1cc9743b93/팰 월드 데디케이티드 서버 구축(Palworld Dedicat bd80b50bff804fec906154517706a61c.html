서버 요구 사항

  • CPU : 4 Core (권고)
  • Memory : 16GB (최소) / 32GB(권고)

방화벽 설정

방화벽 개방

  • 먼저 포트를 열어주기 위해 방화벽 설정

window 키 + S → ‘powerShell’ 검색 → 꼭 ‘관리자 권한으로 실행’

  • 아래 명령어 2개를 순서대로 입력할 것
New-NetFirewallRule -DisplayName "Palworld Server" -Direction Inbound -LocalPort 27015,27016,25575 -Protocol TCP -Action Allow
New-NetFirewallRule -DisplayName "Palworld Server" -Direction Inbound -LocalPort 27015,27016,25575,8211 -Protocol UDP -Action Allow

  • 방화벽 상태 확인

window 키 + S → ‘방화벽 상태 확인’ 검색 → 고급 설정 → 인바운드 규칙

포트포워딩 설정

포트 포워딩 하는 법
  • iP와 게이트웨이 확인

게이트웨이 주소를 인터넷 주소 입력하는 곳에 입력하면 자신이 사용하고 있는 공유기로 접속이 가능하다.

  • 공유기 설정 확인
  1. 포트포워드 설정 클릭
  1. 내부 IP주소 할당, 프로토콜 설정, 외부 포트 설정, 내부 포트 설정 후 적용
  1. 3번처럼 올바르게 적용이 되었는지 확인하면 끝

설치 진행

서버 구동기 설치
  • Dedicated Server 구동기 설치
  1. 라이브러리 검색 창에 Dedicated Server를 찾은 후 설치

  1. 설치 후 실행(옵션은 무엇으로 하든 상관 없음)

  1. 콘솔 창이 열리고 파일을 생성 중이니 5초 정도 후에 창을 닫을 것.

  • 서버 파일 상세 설정
  1. Dedicated Server의 로컬 파일 열기

  1. DefaultPalWorldSetting.ini 파일을 우클릭 하여 편집

  1. 드래그 한 범위 시작부터 끝까지 복사

  1. PalWorldSettings.ini 파일을 똑같이 우클릭 후 편집으로 열기

  1. 파일을 열면 아무것도 없는 것이 정상, 거기에 복사한 내용을 붙여 넣으면 끝

멀티쓰레딩 설정

멀티쓰레딩 옵션 추가
  1. 스팀 라이브러리에서 Dedicated Server의 속성 클릭

-useperfthreads -NoAsyncLoadingThread -UseMultithreadForDS
  1. 표시된 부분에 명령어 복사 후 속성에 입력한 뒤에 따로 저장 할 필요 없이 속성 창 종료하면 설정 완료

마무리

게임 실행 후 확인
  1. 게임을 실행한 후에 ‘멀티 플레이 참가하기(전용서버)’를 선택한다.

  1. 표시된 부분처럼 서버를 만든 사람(호스트)의 아이피를 입력 후 접속
    (ex. 0.0.0.0:8211, :8211 까지 꼭 붙여 주도록 하자)
