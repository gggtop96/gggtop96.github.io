서버 요구 사항

  • CPU : 4 Core (권고)
  • Memory : 16GB (최소) / 32GB(권고)

방화벽 설정

  • 방화벽 개방

  • 먼저 포트를 열어주기 위해 방화벽 설정

window 키 + S → ‘powerShell’ 검색 → 꼭 ‘관리자 권한으로 실행’

  • 아래 명령어 2개를 순서대로 입력할 것
New-NetFirewallRule -DisplayName "Palworld Server" -Direction Inbound -LocalPort 27015,27016,25575 -Protocol TCP -Action Allow
New-NetFirewallRule -DisplayName "Palworld Server" -Direction Inbound -LocalPort 27015,27016,25575,8211 -Protocol UDP -Action Allow

  • 방화벽 상태 확인

window 키 + S → ‘방화벽 상태 확인’ 검색 → 고급 설정 → 인바운드 규칙

포트포워딩 설정

  • iP와 게이트웨이 확인

게이트웨이 주소를 인터넷 주소 입력하는 곳에 입력하면 자신이 사용하고 있는 공유기로 접속이 가능하다.

  • 공유기 설정 확인
  1. 포트포워드 설정 클릭
  1. 내부 IP주소 할당, 프로토콜 설정, 외부 포트 설정, 내부 포트 설정 후 적용
  1. 3번처럼 올바르게 적용이 되었는지 확인하면 끝

설치 진행

  • 서버 구동기 설치
스팀 CMD 설치 : https://developer.valvesoftware.com/wiki/SteamCMD:ko

다운로드 후 압축 파일을 원하는 폴더에 이동

  • 압축 파일 풀기

  • bat 파일 생성

텍스트 파일 2개를 생성 후 파일 명을 각각 “install.bat”, “Start.bat”로 설정

  • install.bat 파일 편집

install.bat 파일을 편집 하여 아래의 명령어를 복사 후 붙여놓고 저장

steamcmd.exe +login anonymous +app_update 2394010 +quit

  • Start.bat 파일 편집

Start.bat 파일을 편집 하여 아래의 명령어를 복사 후 붙여놓고 저장한다.

## 서버 업데이트 구문 ##
steamcmd.exe +login anonymous +app_update 2394010 +quit

## 서버 실행 구문 ##
cd .\steamapps\common\PalServer
start PalServer.exe -log -nosteam -useperfthreads -NoAsyncLoadingThread -UseMultithreadForDS EpicApp=PalServer

  • install.bat 파일 실행

install.bat을 실행하면 참고 이미지의 왼쪽처럼 콘솔 창이 뜨며 필요한 파일을 다운로드

  • Start.bat 파일 실행

Start.bat 파일을 실행하면 특별히 출력되는 메세지는 없으나 필요한 파일을 생성하기 위한 작업이니 5초 정도 기다린 후 창을 닫아주도록 한다.

  • Configure 파일 만들기

참고 이미지의 순서대로 ‘DefaultPalWorldSettings.ini’ 파일의 [/Script/Pal.PalGameWorldSettings] 부분부터 끝까지 내용을 복사한다.

‘DefaultPalWorldSettings.ini’ 파일에서 복사한 내용을 ‘PalWorldSettings.ini’ 파일에 붙여 넣고 저장을 하면 기본 설정은 완료되었다.

  • 월드 설정

‘PalWorldSettings.ini’ 파일을 자세히 보면 월드의 세부 설정을 할 수 있는 매개변수들이 존재 하는데 필요 하에 이 부분들을 수정하면 쾌적한 플레이를 할 수 있을 것이다.

특이사항

  • Dedicated Server 월드 설정 이슈 사항

2024.02.15 기준 아직까지 Dedicated Server는 월드 설정 중 BaseCampWorkerMaxNum의 매개변수를 수정해도 적용 되지 않는 버그가 존재함

해결방법

  • 해당 링크를 통해 WorldOption.sav 파일을 생성

  • WorldOption.sav 파일 추가

참조된 이미지의 파일 이름은 이해를 돕기 위해 ‘SaveFile’ 로 되어 있지만 원래는 ‘5CD353B~~~~~~~’ 이와 같은 명칭으로 되어 있을 것이다.

해당 파일 안에는 ‘Level.sav’, ‘LevelMeta.sav’ 파일이 기본적으로 들어가 있을 것이고 서버를 실행 후 캐릭터를 생성 했다면 ‘Player’ 폴더가 생성 되어 있을 것이다.

하지만 지금 작업에 중요한 것은 아니고 해당 경로에 링크를 통해 생성한 ‘WorldOption.sav’ 파일이 제대로 들어가 있는지 확인을 한 후 서버가 실행하고 있었다면 재부팅 후 정상적으로 적용이 되었다면 참고 사진처럼 적용이 되어 있는 것을 확인 할 수 있다.
